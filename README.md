# tech-docs
using mkdocs to build my GitHub Page from .md

how to run in local

```shell
# install dependency
brew install python
brew unlink pythoon && brew link python
pip3 install mkdocs
pip3 install mkdocs-material
pip3 install mkdocs-bootswatch
pip3 install mkdocs-encryptcontent-plugin

# run 
mkdocs serve
```
